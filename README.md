# Customer Segmentation Using K-Means Clustering
# Project Overview

This project focuses on segmenting customers based on purchasing behavior using K-Means Clustering. The goal is to help the marketing team better understand customer groups and design targeted marketing strategies.

# Business Problem

The marketing team lacked visibility into different types of customers purchasing from the website, making it difficult to create targeted campaigns and personalized promotions.

# Objective

Identify distinct customer segments based on behavior

Analyze differences between customer groups

Provide actionable recommendations for marketing strategy

# Tools & Technologies

SQL

K-means clustering

SAS


# Data Preparation

Performed data cleaning and preprocessing

Engineered features such as:

Purchase frequency

Average order value

Customer engagement metrics

# Methodology
1. Exploratory Data Analysis (EDA)

Analyzed customer purchasing behavior

Identified trends and distributions

2. Feature Engineering

Created meaningful variables to improve clustering performance

3. K-Means Clustering

Applied K-Means clustering algorithm

Used the Elbow Method to determine the optimal number of clusters



# Key Insights

Identified high-value loyal customers with frequent purchases

Identified price-sensitive customers driven by promotions

Identified low-engagement customers with growth potential

# Business Recommendations

Target high-value customers with loyalty programs

Use discounts strategically for price-sensitive segments

Increase engagement through personalized marketing campaigns
